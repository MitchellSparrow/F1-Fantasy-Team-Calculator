Formula One betting data analysis

The following page provides the statistics of the current betting odds of the Formula One drivers and constructors. The table below also shows the driver / constructor current price, as well as their average points that they have achieved for the current season. There are also a few columns which represent the current driver / constructor price rank, points rank and odds rank.

The green highlighted values indicate that the odds rank is currently better than the current points or price rank. A red highlighted value indicates that the odds rank is currently worse than the current points or price rank. Currently all of the odds are collected from SkyBet, and can be found here. In future we hope to add more betting site data to this page. For more information on how this page works and how to use the data, please refer to the How It Works page.
